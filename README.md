# Bad Bug

## Development

You will need to create a `.env` file to connect to the correct website. See `.envexample`.

Run `npm install` to install node modules.

Run `npm start` to start locally.

## Docker Build

You can also build the Docker image. For example, run `docker build . -t datjuanitagurl/bad-bug` to build the image. Then run `docker container run datjuanitagurl/bad-bug:latest` to run it.
